# Renewcast landing page

A bespoke one-page pitch built for Renewcast srl (Rome, Italy), booth 951. The page opens with the 60/30/10 orchestration frame applied to Renewcast's digital-twin-per-asset pipeline, anchors a matching engagement (Feeld) at the price shape their seed-plus stage can absorb, and closes with a 30-minute call with Matt Creamer.

The page uses Renewcast's own blue-green palette (teal #0E8A7A, deep navy #0B3B5C, light accent #6DC7B4) and Inter as the type system, matching the HubSpot-hosted live site. The hero includes a custom forecast curve panel (scheduled vs actual generation) to echo the product surface Renewcast's technical buyers already live inside.

Voice follows `reference/voice-rules.md`: no em dashes, no antithesis patterns, no performed questions, no hype vocabulary. Eduba branding is confined to the footer line. NLP Logix is mentioned once with the required credibility stats ("in machine learning since 2011", "over 150 data scientists"). Ethics Engine is not mentioned (not the right paper for a commercial technical buyer). No pricing is surfaced.

Intended URL path: /for/renewcast

- `index.html` - the page
- `styles.css` - bespoke stylesheet, brand tokens from `brand-pack.json`
